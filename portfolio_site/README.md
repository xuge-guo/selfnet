
# Xuge Guo · Robotics Portfolio

A single-file personal site to showcase robotics projects (quadruped locomotion 0.40 m/s, MuJoCo simulation, RL, hardware).

## Files
- `index.html` — the site (open directly or deploy)
- `Xuge_Guo_Resume.pdf` — your résumé (already included/copied if you provided it)
- `preview.jpg`, `thumb.jpg`, `makecu.jpg` — optional images (placeholders included)
- `report.pdf` (optional) — project report
- `journey.mp4` (optional) — local video fallback (online link is in the page)

## Quick Deploy

### Option A — Netlify (drag & drop)
1. Go to https://app.netlify.com/drop
2. Drag the whole folder onto the page
3. Done (a public URL is generated)

### Option B — Vercel (Git)
1. Create a GitHub repo, push this folder
2. Go to https://vercel.com/new → Import the repo
3. Build/Output: static (no config needed)

### Option C — GitHub Pages
1. Create a repo, commit the files to `main`
2. Settings → Pages → Deploy from branch → `main` / root
3. Your site is live at `https://<username>.github.io/<repo>/`

## Customize
- Update links (video/GitHub/LinkedIn) inside `index.html`
- Replace images with your real photos (keep the same filenames)
- If you add a `report.pdf` or `journey.mp4`, place them in the same folder

## Local Preview
Just open `index.html` in your browser.
